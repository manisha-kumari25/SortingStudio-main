# Sorting Studio

An interactive sorting algorithm visualizer built with React, TypeScript, and Vite. Watch popular sorting algorithms in action with customizable animations and multiple visualization themes.

## Features

- **6 Sorting Algorithms**: Bubble, Insertion, Selection, Merge, Quick, and Radix sort
- **Interactive Visualization**: Step-by-step animation of each algorithm's execution
- **Race Mode**: Compare multiple algorithms side-by-side
- **Multiple Themes**: Balloons, Bars, Lines, Lollipops, and Fun visualizations
- **Adjustable Speed**: Control animation speed in real-time
- **Code Mode**: View and study the algorithm implementations

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Motion (Framer Motion)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
